Getting Started in Jupyter for Python

Date Mon 03 October 2016 Tags Python / Jupyter / Programming

Python is quickly growing acceptance in the scientific and engineering community as a computational tool due to its open source nature, meaning it can be installed anywhere for free, and quickly expanding toolkit. With the scientific packages being developed at a rapid pace it is quickly approaching parity with established commercial tools such as Matlab in some domains. My reasons for moving to Python have previously been discussed and continue to grow as I discover greater and greater power in using it for my primary language. My other getting started entry may also appeal to help individuals although the Plotting with matplotlib is likely the best way to start from zero knowledge.

On top of Python being such a powerful language, the Jupyter interface creates a powerful environment in which to work to develop programs and perform analysis. For more substantial code, you will likely move to add on another editor such as Atom or Spyder, Jupyter is a both gentle way to try out the language and experiment with code while you learn python tools.

Like any freeware, it is held back by the lack of an advertising budget (granting visibility) awareness, and fear of the installation process, which can be daunting for some open source projects expecting a strong command of Linux. These fears are unfounded as there are easy ways to install and run Python on your choice of operating system (MacOS X, Windows, Linux). For the novice getting started, I advocate using the Anaconda installer. It will vastly simplify the process and assure you have what you need to get going. I personally use MacPorts, but this doesn't apply broadly. For my Linux and Windows installations, I indeed use Anaconda.

Install Jupyter Notebook

To start, simply see the Anaconda Install page. They should be adequate to get all the files you need in place.

Run Jupyter Notebook

Windows User

  1. Install Firefox or Chrome, although the Edge browser in Windows 10 seems to work ok.
  2. Run the Anaconda Command Prompts in your Start Menu (you will have to select All Apps then look under Anaconda).
  3. Use the Windows cd command to change to your project directory
  4. Type conda install jupyter to install Jupyter (not every time)
  5. Type conda update --all to make sure conda is up to date (occasionally)
  6. Type jupyter notebook from a terminal prompt.

Mac user

  1. Install Firefox or Chrome.
  2. Run the Mac Terminal (in the Applications/Utilities Folder)
  3. Use the cd command to change to your project directory
  4. Type conda install jupyter to install Jupyter (not every time)
  5. Type conda update --all to make sure conda is up to date (occasionally)
  6. Type jupyter notebook from a terminal.

Linux user

  1. From your repository (presuming you've already cloned it using the command line)
  2. Type conda install jupyter to install Jupyter (not every time)
  3. Type conda update --all to make sure conda is up to date (occasionally)
  4. Type jupyter notebook from a terminal.
